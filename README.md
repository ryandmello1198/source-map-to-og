# Source Map to OG Mapper

Uses [`source-map`](https://github.com/mozilla/source-map) module

## Usage

##### Clone this repo 
##### Install the dependencies
     npm install
##### Run index.js with appropriate flags
      node index.js -f filename.js.map -l 1 -c 268167
      
## Flags

    -f, --file <filename>
    -l, --line <line>
    -c, --column <column>
    -h, --help
    -v, --version
